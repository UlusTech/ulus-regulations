[![Language: TR](https://img.shields.io/badge/Language-TR-red.svg)](./README.md)
[![Language: EN](https://img.shields.io/badge/Language-EN-blue.svg)](./README.en.md)

Ulus consists of units and works via regulations. Regulations are files that standardise topics like work ethics of Ulus, operation of projects. If you are interested in Ulus, this is the place you must come first.

Regulations are regulated by Üst Yönetim Kurulu which manages Ulus and these regulations determine everything about Ulus. Everything you see in these files are a cultural structure accepted by a group of people. Regulations are not written to control this culture, instead this culture is made into regulations.

Üst Yönetim Kurulu uses [Git](https://git-scm.com/) for regulations.
Coming from this; with this project, Ulus Tech provides the regulations through GitHub, aiming to improve accessibility because of developers' requirements. Originally, regulations will be provided through the website of Ulus with an interface and this exact project will be available to download through the website.

This source is just a project of a unit. For the sake of data reliability and accuracy, the website of Ulus must be used.
